- **[Controllers](/docs/nestjs-crud-automator/core-concepts/controllers)**: Automatic controller generation with ApiController
- **[Services](/docs/nestjs-crud-automator/core-concepts/services)**: Service layer with ApiService and ApiServiceBase
- **[DTOs](/docs/nestjs-crud-automator/core-concepts/dtos)**: Automatic DTO generation and customization
- **[Metadata Storage](/docs/nestjs-crud-automator/core-concepts/metadata-storage)**: How metadata is stored and retrieved
